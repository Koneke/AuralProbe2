Main window

Overview

The main window is split into four sections:

  • Menu bar: The menu bar at the top shows commonly used menu items.
  • Categories: The categories on the left side show the sample categories, with the number of samples in that category in brackets. Categories can be modified in the configuration dialog.
  • Samples:To the right of the categories, a list of all samples in the current category.
  • Status bar: The status bar at the bottom shows information about the current operation, as well as sample information when a sample is highlighted, such as file path, bitrate, format, duration etc.

Menu bar

The menu bar has the following items:

  • Rescan: Manually rescan search folders for audio samples. If you make changes to your sample collection, run this to ensure Aural Probe stays up to date.
  • Favorites: Toggle favorites on/off. When enabled, only samples marked as favorites are shown. When disabled, all samples are shown.
  • Open...: Open an Aural Probe favorites collection.
  • Save...: Save the currently marked favorites to an Aural Probe favorites collection.
  • Reset: Reset the currently marked favorites.
  • Play/Stop: Play or stop the highlighted sample.
  • Config...: Display the configuration dialog.
  • Help: Display this help file.
  • About: Display the about dialog.
  • Volume: Adjust volume for sample playback.
