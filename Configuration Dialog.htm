Configuration dialog

General

  • Sample Display Size: The display dimension of each sample. Use small values to display lots on screen at once.
  • Always on top: Force Aural Probe to stay on top of other applications, even when not in focus. Useful for multiple drag-and-drop operations.
  • Prompt when rescanning folders: When checked, Aural Probe will always prompt the user for confirmation before scanning search folders. If you have a small sample collection, you may want to disable this.
  • Include file paths when searching: When checked, Aural Probe will use file paths as well as file names to match search criteria. For example, the sample "C:\Samples\Kicks\01.wav" would only match the search term "kick" if this is checked.
  • Autoplay: When checked, Aural Probe will auto-play samples sequentially, repeating them a specified number of times before advancing to the next sample. Playback can be stopped at any time by pressing "Stop" in the menu bar.

Categories

  • Categories list: Displays a list of categories, along with their search criteria. Click one to modify it.
  • Name: The category name. Can be used to create new categories, or update existing categories.
  • Search criteria: The category search criteria. Case insensitive. Separate search terms with commas, for example: "kick,bd,bassdrum,bass drum"
  • Use regular expressions: Should the search criteria use regular expressions? In this case there is only one search term. Case insensitive.
    Examples:
    ".ogg$" would match all samples ending in .ogg
    "k(i)*ck[0-9]+" would match kck0, kick10, kick999 but not kck_9 or kickdrum
    For full syntax visit: https://msdn.microsoft.com/en-us/library/az24scfc%28v=vs.110%29.aspx
  • ↑: Move the selected category up in the category list.
  • ↓: Move the selected category down in the category list.
  • Insert: Insert a new category into the category list with the current category name and search criteria.
  • Replace: Replace the existing category in the category list with the current search criteria.
  • Delete: Delete the existing category in the category list.

Search folders

A list of all folders to search. Can be one or more local or network folder paths, one per line, for example: 
C:\Samples
E:\DrumSamples
\\music\samples

Default favorites folder

The default folder to save Aural Probe favorites. Can be left blank.

File types

A list of sample file types to include. Uncheck boxes to exclude samples of that file type.

Sound output

The sound playback device. Useful for musicians who often have multiple playback devices.
